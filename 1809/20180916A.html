【归纳】如何CSS实现各种布局

广宇 2018-9-16 CSS

再编辑于 2018年11月18日 19:18:35

这篇博客是我在9月16号的时候，刚学CSS的时候写的。当时写的时候什么都还不懂，现在准备写关于CSS的押题，正好涉及到居中布局这一块，就回头来看了。虽然以前写的并没有什么错误，但是感觉还是有些地方是可以完善的，这次回头就把这篇博客做得更好一些吧。

我开博的时间在今天正好达到了两周，这篇博客也将是我要写的第七篇博客。这次博客主要的内容就是对 CSS 的布局方式和方法做一个简单的归纳。布局是 CSS 中一个重要部分，包括常用的水平居中、垂直居中方法，以及左右布局、左中右布局的多种实现方式。通过这次归纳，希望能给自己一些新的启发。

# 左右布局

css实现左右布局显示是前端进行页面设计的基础，也是全面了解并学习css一个很好的切入点，因为其中会涉及到对许多css基础点的认知。实现css入门，理解左右布局的实现方式是必经之路，同时也能使项目中涉及前端编码的部分受益。

在课上实现左右布局的方法是使用 float 实现左右布局。float属性是css中关于布局的一个关键属性，其意为将该块状区域脱离父级标签的文档流，left属性值使该区域向父级标签区域的左侧边界放置，right属性值使该区域块向父级标签的右侧边界放置，如是利用该属性可以实现左右布局。 float属性属于布局属性，其中有着很多重要应用。

下面的就是一个左右布局的样例：

此div实现左布局
此div实现右布局
#wai{
    overflow:hidden;
    margin: 0 auto;
}
.divleft{ 
    float:left;
    text-align:center;
}
.divright{ 
    float:right;
    text-align:center;
}

# 居中布局

水平居中和垂直居中在编写HTML5静态页面过程中经常遇到，现在就来总结一下对于不同的元素如何实现居中布局的方法。之前就是这些地方没有弄清楚，关于居中布局也是面试可能会考的地方，所以这里我重新整理一下，对之前的内容进行一些完善。

# 1、flex布局居中

使用flex布局可以轻松将父元素内部的元素实现完美居中（即水平方向与垂直方向都居中）：

.container{
    display:flex;
    align-items:center;
    justify-content:center;
}

    display:grid;

    justify-content:center;

# 3、display:table-cell

在 chrome、firefox 及 IE8 以上的浏览器下可以设置块级元素的 display 为 table-cell，激活 vertical-align 属性。这个方法是在flex布局流行前的常用方法，但注意 IE6、7 并不支持这个样式。

.wrapper > .container{
    display:table-cell;/*IE8以上及Chrome、Firefox*/
    vertical-align:middle;/*IE8以上及Chrome、Firefox*/
}
.wrapper{
    diaplay:table;
    text-align:center;
    width:100%;
}

使用这种方法是借助了vertical-align:middle可以使得表格元素与行级元素中线对齐的特性。通过设定display:table-cell，.container元素会被当作表格中的格子来渲染。

但是这样做得话.container的宽度就会收缩，因为其已经失去了块级元素的特性。因此需要设置.container的父元素.wrapper为diaplay:table，宽度自适应，并且text-align:center，这样就实现了.container中的元素的完美居中。

# 4、奇葩方法

利用vertical-align的特性实现垂直居中，这一点在写IFC的那篇博客里也提到过一些，这种方法可以使行级元素垂直居中，也是利用了vertical-align:middle居中对齐的特性。但是这种方法比较麻烦，不推荐使用：

.container{
    width:200px;
    height:200px;
}
.container::before{
    content:'';
    display:inline-block;
    width:0;
    height:100%;
    vertical-align:middle;

# 5、绝对定位

适用于绝对定位的场景，是传统方法：

.container{
    position:relative;
}
.child{
    position:adsolute;
    top:50%;
    left:50%;
    transform:translate(-50%,-50%)

# 6、单行文本垂直居中

在父元素高度没写死的情况下，设置父元素的上下padding相等就可以实现元素内的文字垂直居中对齐。

.container{
    padding:10px 40px;
}
1
2
3

如果父元素确定高度的单行文本通过设置 line-height 与 height 相同的值即可完成。此时80px即为父元素宽度，无需再去设定height的值了：

.wrap{
    line-height:80px;
}

line-height 与 font-size 的计算之差，在 CSS 中称为「行间距」，分为两半，分别加到一个文本内容的顶部和底部。但是当文字内容的长度大于块的宽时，就会有内容脱离块。

# 7、水平居中

# 1)行级元素

对为块级元素的父元素使用，能让内部的匿名行盒（文字）、行内元素（span）、inline-block元素在父元素中水平居中：

.container{
    text-align:center;
}
1
2
3

# 2)块级元素

块级元素的水平居中text-align:center就不起作用了，这时候分两种情况：

# (1)定宽块状元素

可以通过设置左右 margin 为 auto 即可实现。

div{
    margin:0 auto;
}

# (2)不定宽块状元素

可以使用如下方法实现：

# 方法一

加入 table 标签； （其实就是设置父元素display:table-cell，不过这种方法兼容性更好就是了）

<table>
    <tbody>
        <tr>
            <td>
                <p>设置我所在的div容器水平居中 </p> 
            </td>
        </tr>
    </tbody>
</table>

这是利用table标签的长度自适应性——即不定义其长度也不默认父元素body的长度（table其长度根据其内文本长度决定），因此可以看做一个定宽度块元素，然后再利用定宽度块状居中的margin的方法，使其水平居中。

# 方法二

设置 display:inline 方法，然后 text-align:center 实现。与第一种类似，显示类型设为行内元素；

.container{
    text-align:center;
}
.container div{
    margin:0;padding:0; display:inline;
}
