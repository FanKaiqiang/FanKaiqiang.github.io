【归纳】icon的各种做法

广宇 2018-9-27 CSS

这篇博客是对我前一段时间关于icon学习的一次归纳。之前学完之后没有写博客，只做了笔记。最近决定坚持写些博客，因此我要把之前没写成博客的知识补回来，顺便复习一下。这次归纳的知识点是icon的各种做法。icon在网站设计中是不可或缺的，如何多快好省地把icon做出来也是非常重要的一个知识点。接下来我就将我上次学习的一些做法逐条列出。

# 用img显示icon

通过 img 元素，可以向网页中嵌入一幅图像，由此就可以达到显示icon的目的。下面先介绍一下如何将需要的icon从图像文件中提取出来的方法。

# PSD文件

PSD文件用photoshop打开，是PS专有的位图文件格式。在图像制作完成后，通常需要转化为一些比较通用的图像格式（如：jpg、png、tiff、gif格式等），以便于输出到其它软件中继续编辑。使用PS从PSD中导出icon的方法主要有以下几步：

  1. 选中图层：Duplicate Layer（复制图像） → 导出
  2. 裁剪图层：Image → Trim（裁剪） → 裁剪完成
  3. 修正宽高：Image → Canvas Size → 修改宽高
  4. 导出为PNG：File → Export → Quick Export as PNG 或 Save for Web（Legacy）

# PNG文件

PNG大家想必都很熟悉，但是要将需要的icon从PNG中取出恐怕就没有PSD那么方便了。这时还要用到PS，将需要的部分裁剪出来，再进行处理。主要有以下几个步骤：

  1. 选中PNG图片需要导出区域
  2. 剪切图片：Image → Crop
  3. 魔法棒：选中需要的区域 → 右键 → Select inverse
  4. 删除选中区域
  5. 裁剪图层：Image → Trim（裁剪） → 裁剪完成
  6. 修正宽高：Image → Canvas Size → 修改宽高
  7. 导出为PNG：File → Export → Quick Export as PNG 或 Save for Web（Legacy）

使用img元素显示的icon，可以自适应宽高，图片默认会保持比例，在修改宽高时不需要同学修改 width 和 height 。

# 用background显示icon

与使用img元素不同，用background就是通过显示背景图片的方法将icon的效果显示出来。如何将icon图像文件提取出来的方法前文也都提到了。下面就来看一下一个使用background显示icon的示例：

<div class="icon">
</div>
<style>
.icon{
    margin:5px 25px;
    border:1px solid red;
    display:inline-block;
    width:100px;
    height:100px;
    background:transparent url(icon.jpg) no-repeat;
    /* 背景颜色透明，图像不平铺 */
}
</style>

# 雪碧图 CSS Sprite

CSS雪碧图 即CSS Sprite，也有人叫它CSS精灵，是一种CSS图像合并技术，该方法是将小图标和背景图像合并到一张图片上，然后利用css的背景定位来显示需要显示的图片部分。

要想获取CSS Sprite，可以借助CSS Sprite generater。将需要的icon上传，经过CSS Sprite generater的处理，可以得到需要的雪碧图及css、html的代码。将他们加入到网页上就可以完成icon的显示。

使用CSS Sprite的所有div都拥有整张图片作为背景，但只露出部分作为icon。

# font法（字体法）显示icon

在html中，可以用一些以&开头的字符串来表示其他的字符，这些字被称为html实体，如 &nbsp;、&#160; (都是空格)等。

font法即是将不表示任何字符的html实体在自定义的字体中用来表示icon字体。接下来将要使用Iconfont-阿里巴巴矢量图标库，借助这个网站来介绍使用font法显示icon的两种方法。

# Unicode法

此方法通过将字体代码copy至CSS，再将该CSS应用至html中即可。字体代码示例如下：

@font-face {
  font-family: 'iconfont';  /* project id 838980 */
  src: url('//at.alicdn.com/t/font_838980_xn8qp8g11lk.eot');
}

# SVG icon（symbol引用法）

SVG 是一种可缩放矢量图形（英语：Scalable Vector Graphics，SVG）是基于可扩展标记语言（XML），用于描述二维矢量图形的图形格式。可以在iconfont.cn中获得使用SVG显示icon的帮助文档，在这个网站可以很方便地用此方法来显示icon。根据帮助文档得知，全程主要有以下几个步骤：

# 第一步：拷贝项目下面生成的symbol代码：

//at.alicdn.com/t/font_8d5l8fzk5b87iudi.js

# 第二步：加入通用css代码（引入一次就行）：

<style type="text/css">
    .icon {
       width: 1em; height: 1em;
       vertical-align: -0.15em;
       fill: currentColor;
       overflow: hidden;

# CSS icon

使用纯CSS制作icon，虽然很麻烦，但是这种方法可以锻炼CSS的能力。在这个网页：https://cssicon.space/，可以看到很多由纯CSS制作的icon，从中可以学到许多CSS的技巧。

这次博客大概就将现在制作icon的所有通常方法做了一个概括。可以看出，在前端工作中，合理使用工具是及其重要的，在某些方面有人工完成可能会很麻烦，但是只要能够掌握这方面的辅助工具，完成起来便事半功倍了。
