# Array()

这节课是从对标准库的归纳入手的，标准库是什么？标准库指的是js的标准内置对象，是js这门语言本身初始时提供的在全局范围的对象，它们存在于window在堆内存引用的Hash中。其中包含：Object()、Number()、Boolean等，这些在之前的博客都谈到过一些，这回的主题是之前没有讲过的对象，比如Array()。

Array()是对于构造数组的全局对象。在使用这个对象时需要了解Array()的一些特点，比如：

① 下面两种声明数组的方式是等价的：

let fruits = ["Apple","Banana"]
let fruits = new Array('Apple','Banana')
1
2

② var a = Array(3) 代表长度为3的数组，而不是内容为“3”的数组。在这个数组中，length=3。但是需要注意的是，作为key的0、1、2并没有被存入数组中。也就是说，数组的内容是undefined，但是长度为3。数组设一个参数时，参数为数组的长度。

③ 数组设两个参数时，参数被设为数组中的项：

var a = Array(3,3)
a = [3,3]
1
2

④ 以下两种声明数组的效果是相同的：

var a = Array(1,2,3)
a = [1,2,3]
1
2

⑤ 声明数组时，加new与不加new对Array都是一样的：

var a = new Array(3)
var a = Array(3)
1
2

# 数组与伪数组

刚才讲到了数组的声明。可以知道，数组是对象的一种，但是究竟什么是数组呢？可以这样理解，数组是按次序排列的一组值。在JS中，Array构造出来的对象就是数组。声明一个数组：

a = [1,2,3]
a.xxx = 'xxx'
a.yyy = 'yyy'
a.length        // 3
a.__proto__     // Array.prototype
1
2
3
4
5

可以像对象一样给数组进行赋值，事实上，数组就是对象的一种。但是需要注意的是，在给数组添加key不为数字的值时，并不会对数组的length产生影响，及数组的长度不受影响。不过当使用key in a仍然可以遍历到a中的'xxx'、'yyy'等key。

数组与对象的根本区别是因为数组具有数组的原型，普通的对象没有。数组之所以为数组，是因为把它当做数组。事实上，就刚才数组的表现来看，完全可以把数组当做普通对象来使用。

反过来，可以试着将普通对象当做数组来用，这种对象具有与数组很相似的特征，将之称为伪数组。伪数组像数组一样，以数字为key，并且具有length属性。比如arguments就是一种代表函数传入参数的伪数组：

f(1,2,3){
    arguments   // {1,2,3} 
}

# 数组的API

在数组的原型中提供了许多可以对数组操作的属性，将之称为API。借助这些API，能够对数组进行许多操作，本节就对一些数组操作API进行讲解。

# arr.forEach()

forEach() 方法对数组的每个元素执行一次提供的函数。该方法会接受函数x为参数，同时遍历这个数组，对数组的每一项都会传入函数，再对函数x进行调用。

传入a.forEach的函数接受的第一个函数为数组a的值，第二个参数为数组的key，下面是第三个参数为a本身。大概是以这个格式进行表示：

arr.forEach(function(value,key,arr){...})
1

同时，以下两种使用forEach的方法是等价的：

a.forEach(function(...){...})
a.forEach.call(a,function(...){...})
1
2

下面是一个forEach使用的示例：

var array1 = ['a', 'b', 'c'];

array1.forEach(function(element,key,a) {
  console.log(element);
  console.log(key);
  console.log(a);
});

// > "a"
// > 0
// > Array ["a", "b", "c"]
// > "b"
// > 1
// > Array ["a", "b", "c"]
// > "c"
// > 2
// > Array ["a", "b", "c"]

# arr.sort()

sort() 方法用内置排序算法对数组的元素进行排序，并返回数组。内置的排序方法一般是快速排序。下面是几个使用示例：

a = [5,6,3,4,1,2]
a.sort()
a      //  [1,2,3,4,5,6]
1
2
3

还可以对排序的顺序进行决定，默认是正序排列，通过给sort()传入一个函数参数可以设置排序的准则，假设 x<y，当return的值为负时，即为正序排列，否则为反序排列：

a.sort(function(x,y){return x-y})  //正序排列
a.sort(function(x,y){return y-x})  //反序排列
1
2

借助这个函数，还可以进行一些其他特殊的排列：

a = ['马云','马化腾','李彦宏']
hash = {
    '马云':167,
    '马化腾':376,
    '李彦宏':228
}
a.sort(function(x,y){hash[x]-hash[y]})
a   //   ['马云','李彦宏','马化腾']

# arr.join

join() 方法将一个数组（或一个类数组对象）的所有元素连接成一个字符串并返回这个字符串。可以借助这个方法可以用于将数组转化为字符串，下面是一些使用示例：

a = [1,2,3]
a.join('a')     //  "1a2a3"
a.join()        //  "1,2,3"
a.toSting()     //  "1,2,3"
a + ','         //  "1,2,3"

# arr.concat()

concat() 方法用于合并两个或多个数组。此方法不会更改现有数组，而是返回一个新数组。下面是一个使用示例：

 a = [1,2,3]
 b = [4,5,6]
 a + b           //   "1,2,34,5,6"
 a.concat(b)     //   [1,2,3,4,5,6]
 var c = a.concat([])
 c === a         //   false

可见concat()方法是将两个数组连接为一个新的数组。如果连接的数组是空数组，生成的数组也不与原数组相等，这是对象的特性导致的。

# arr.map()

map() 方法创建一个新数组，其结果是该数组中的每个元素都调用一个提供的函数后返回的结果。map()方法功能与forEach()相似，map除了遍历数组之外，还会把函数对数组的操作的结果收集起来，并返回一个新的数组。如：

a.map(function(x,y){return x+y})   //[1,3,5]
a.map(x => x+y)   //[1,3,5]
1
2

map是映射的意思，即是对a的值进行映射，并不改变a数组的原值。之前讲的sort()方法是会改变排序数组的原值的，是为了节省空间才这样做的。

# arr.filter()

filter() 方法创建一个新数组, 其包含通过所提供函数实现的测试的所有元素。该方法会筛选符合设定条件的值，并将值推入到一个数组，并不会对原数组产生影响。下面是一个使用示例：

a = [1,2,3,4,5,6,7,8,9]
a.filter(function(x,y){
    return x<5
})         //   [6,7,8,9]

# arr.reduce()

reduce() 方法对累计器和数组中的每个元素（从左到右）应用一个函数，将其简化为单个值。reduce()会遍历一个数组，每次取一个结果，再把这个结构放在下一个项上，下面是一个例子：

a.reduce(function(sum,n){
    return sum+n
},0)
1
2
3

在这个例子里，sum为每轮操作结果的累积值，n为遍历中的数组的值，0为sum的初始值。利用reduce()方法的特性可以模拟其他方法，比如：

a.reduce(function(arr,n){
    arr.push(n*2)
    return arr
},[])

上面将来一些数组的常用API，事实上数组的API非常多，这里只是讲得很小一部分，我对其他的API也并没有什么了解。比如reverse()方法可以用于颠倒数组中元素的顺序。在“Array - JavaScript|MDN”中，可以查看所有数组操作API。

这些API可以混合使用，下述操作即是先将a以x函数提供的条件过滤，再以y函数对a进行操作。
