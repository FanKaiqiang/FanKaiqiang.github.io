【分析】JS实现排序算法1——冒泡排序

广宇 2018-10-2 JavaScript 算法

自从前几天出差回来，就断了几天的博客写作，同时我的学习规律也被打乱了不少。今天时间也比较晚了，我就写一篇应该不算很长的博客。这次的博客应该是一个系列的，因为我这几天学习了通过JavaScript实现各种算法。学完之后感觉还是需要归纳一下比较好，但是因为内容多，为了归纳地更详细，效果更好，所以我打算写成关于算法的博客系列，方便我巩固这方面的知识。

说到算法，就要先谈谈算法的特征，什么是算法呢？

以下是我在学习笔记中对算法的特征归纳：

  1. 输入：一个算法必须有零个或以上输入量。
  2. 输出：一个算法应有一个或以上输出量，输出量是算法计算的结果。
  3. 明确性：算法的描述必须无歧义。
  4. 有限性：算法必须在有限时间内完成任务。
  5. 有效性：又称可行性。算法中描述的操作都是可以通过计算机完成。

算法作为一个重要知识点，把每种算法都弄明白是很有必要的。通过这次关于算法的博客系列，我将在博客用自然语言解释几种排序的具体排序规则，以及使用JavaScript将几种排序规则实现。本次博客就将冒泡排序的规则以及JavaScript实现方式进行分析。

# 冒泡排序

冒泡排序是可以将数组中的数字由大到小进行排序的一种排序方法。其主要步骤大致如下：

  1. 数组中第1个数字与第2个数字进行比较，如果第1个大于第2个，则两个数字交换位置。
  2. 数组中第2个数字与第3个数字进行比较，如果第2个大于第3个，则两个数字交换位置。
  3. 以这种思路进行排序，直到排到最后一个数字（设为n），此时数组中最大的数字便被排到数组的最后一位，第一轮排序结束。
  4. 第二轮排序开始，重复上一步操作，此时只需要排到第n-1个数字，数组中第二大的数字也被排到数组后方。
  5. 经过n-1轮的排序，每个大数字都被移至小数字的后方，排序完成。

接下来就将对冒泡排序的JavaScript的实现代码进行分析：

function bubleSort(arr){
    for(let i = 0; i < arr.length-1; i++){
        for(let j = 0; j < arr.length-1-i; j++){
            if(arr[j] > arr[j+1]){
                [arr[j],arr[j+1]] = [arr[j+1],arr[j]]
            }
        }
    }
}

其中：

  1. “for(let i=0;i < arr.length-1;i++)”这一段代表排序的轮数，排序需要经过length-1轮的排序，即i=length-1时退出循环。其中length为数组长度。
  2. “for(let j = 0; j < arr.length-1-i; j++)”这一段代表当前被选中排序的数组的下标，每轮排序将会在排到下标为length-1-i时完成该轮排序，即j=length-i-1时退出循环。
  3. 在arr[j] > arr[j+1]时，arr[j]与arr[j+1]交换位置。

冒泡排序的时间复杂度为 O(n^2)，数组长度每增加10倍，排序长度增加100倍。
