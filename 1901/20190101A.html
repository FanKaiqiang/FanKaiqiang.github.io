【开幕】2019-01 的简单计划

广宇 2019-1-2 思考

今天是 2019 年的第一天，也是我大学生涯的最后一年的开始。本来今天我可能已经不打算写博客了，但是在前几天写完几篇博客的任务基本完成之后，又有些失去目标的感觉，今天不知道看什么课程了，要看的内容有点多。

所以虽然前几天才写了 12 月份的总结随笔，但是其中对于一月份的安排并不具体。所以这篇博客的主要目的还是安排安排一月份的工作，顺便给自己的新年学习开个头。

这篇博客是【开幕】类型的博客，因为今天是 2019 年的第一天嘛。不仅是给一月份做计划，展望一下新的一年也是比较重要的。这次的背景音乐就用这首，和开博大吉的那篇博客也算是能够互相呼应了：

从第一篇博客的时间算起，不知不觉距离开始学习前端知识已经到了第五个月了，今天的博客写到了第七十篇。回顾过去的几个月不禁感慨，在七八月份在宁波实习的时候还不知道自己以后要干什么工作，大有失业的危险。虽然五个月之后我仍然是待业青年，但是我比之前的时候还是踏实了几分，现在就算是我没本事、找不到好工作，也比以前有底气了一些。

虽说如此，我心里现在还是发虚的，因为我的毕设课程目前只做了一个「自制 UI 组件」，我真的没想到这个毕设要做掉我一个月的时间。现在想来，我的第四个月的学习效率确实是不怎么好，如果这个月还是这样，恐怕到找工作之前也别想学完全部课程。所以要想在剩下两个月的时间里把课程学完，特别是在元宵节之前，从现在开始的时间必须要好好安排，如果时间不紧凑些的话，到时候就难办了。

促使我今天决定好好把自己要看的课程做个安排是有原因的。在今天之前，我心里还在想看看其他的拓展课程放松放松，毕竟看毕设课程一看就是一整天。今天我发现，我手上的拓展课程也很多，毕设课程还有五个要做的，最后我还要花些时间完善我自己的项目。如果我还要把时间花在犹豫看什么课程好上面的话，那时间就白白浪费了，课程也没看成。所以我现在就干脆决定好接下来应该看些什么，其他的就干脆放着一边，学完这部分课程，再学下一部分课程。如果总是着急课程多看不完，倒不如抓紧时间开始看一些，这样才能更有效解决我的困境。

首先我来捋一捋我有哪些课程想看的：

  • 【毕设】React 通讯系统（12）
  • 【毕设】Vue 搭建社区（12）
  • 【拓展】React 入门（10）
  • 【拓展】TypeScript 入门（8？）
  • 【拓展】图解 HTTP 导读（？）
  • 【深入浅出】ES6（7）
  • 【深入浅出】JS（13）

列出来才觉得有点多啊，看来是刻不容缓了。首先毕设课程是首要完成的任务，两个毕设课程我计划是在一月之内完成，这两个毕设是比较大的毕设。剩下三个我打算在做一个 React、一个小程序、一个 Node.js 的，这三个看起来不用那么多时间，可能要在二月份来做了，所以时间是非常紧迫的。上面的这两个毕设一共 24 节，如果能够争取一天看完一节的内容，也要看到 25 号才算是看完，还不算有的时候可能看不完，比如做自制 UI 组件毕设的时候，就可能要两天才能看一节课。

所以尽量在一月之内看完这两个毕设吧。一月份的话同学都准备回家了，寝室基本就是我一个人了，这下如果自己学习效率低就不能甩锅给同学了，全都要靠自己把握了。一月份我计划 20 号回遂川，那我现在还要十来天要呆在学校，其中大部分时间都是一个人在寝室学习，时间要好好安排。首先是起床，就决定七点半起床，八点钟吃早饭。在八点半开始学习，午觉的话如果要睡也要在两点钟之前醒来，不应该在睡觉和起床之间话费过多的时间。晚上的话尽量十二点前睡觉，这样才能保证第二天起得来，不会在早上的时候过于疲惫。

学习内容方面，首先开始做 React 通讯系统的毕设，计划每天一节。此外在每天如果时间有空余就看一节 React 入门的拓展课程。每天最好是能看一节毕设课加一节其他的某些课程，其他的成的选取上优先选择 React 入门，这些课程是博客题材的主要来源，可以视情况选择学习。

博客方面，算上这篇博客已经有 70 篇博客了。这个月仍然是多多益善，但是时间紧，可能写不了 20 篇，但是我也会尽力的，如果当天的毕设课程看完在晚上还有时间的话，就可以用来写一篇博客。这个月至少要写 15 篇，尽量写到 20 篇。不强求几天写多少篇博客，因为之前几天我发现，只要愿意花时间写，抽出一两天时间用来写博客还是可以整理出很多东西来的，不必每天心里为自己给自己定的博客任务着急，尽快学完毕设才是最重要的。

到这里我就已经基本把事情安排好了。总得来说就是每天「一节毕设 + 一节拓展 + 一篇博客」，其中以毕设课程为主，如果当天毕设课程看完了，再来看拓展课程，或者是写博客。上述计划明天就开始实行，明天尽量把 React 毕设的第一课看完吧，内容有点多，尽力吧。一天如果能花 12 个小时学习难道还看不完 6 个小时的课程吗？只是时间没利用到罢了，这个月既然排除了所有外在因素的干扰，我就得好好利用所有的时间了。最后希望我在新的一年能够顺利吧。
