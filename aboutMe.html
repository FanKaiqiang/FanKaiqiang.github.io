广宇 2019-8-4 思考

你终于来了。

这里是广宇的个人博客，里面放着的都是一些我的学习过程中的一些笔记，偶尔也有一些随笔。除此之外，目前并没有放些其他东西的打算。而且我只是一个普通人，这儿也并不是什么大佬博客，要找大佬的可以离开了😊。这里是我专属的一片空间，不会有任何人来打扰你我之间的交流。

为什么这么说呢？因为本博客目前崇尚的风格是极简，只要能够让我自己看到我自己的文章在网页上，就可以了。当然，如果真的只是那么简单的话，直接把纯文本写在html上，或者是直接在 GitHub 建一个纯markdown仓库不就得了。所以，我还是希望我这儿能够更加精致一些，至少来到这儿的你不会感到惊愕，以至于直接关闭这个网页吧。

而且，这个网站并没有接入百度和 Google 的搜索系统，用搜索引擎在外面确实是搜索不到这个博客网站。至于你的话，我想，要么是不知道从哪儿误打误撞从我的 GitHub 中进入了这里，要么就是认识我的人，知道这儿是我的博客网站。不管怎么样，我想都是对我的博客至少有那么一丝丝的兴趣，才会点开来的。不管是否认识我，通过这里的博客了解我对我都没有什么损害，这里的内容并不会不利于我，你也无法因为我的博客内容而对我造成什么影响。

不过，虽说是你我之间的交流，但是你可以发现我并没有给你留下输出的空间😏。这个博客主题原本是有评论系统的，前几天我申请 LeanCloud 的实名制两次失败了。就想我干脆还是不做这个劳什子评论了，反正平时也没人来，也不会有人来给我打尻。所以你在这有什么想说的话，或者是对某篇文章有什么意见，都没法输出给我了。现在这儿就是我的一言堂，当然你要是硬有什么事要和我说的话，我在下文也留了有邮件，你发邮件来就可以了。

说了这么多，也该开始介绍一下关于我的博客的内容了。我的博客实际上早在去年（2018年）就建立了，那个时候我还在读大四，刚刚决定学习前端方面的知识，希望自己能够在今年（2019年）毕业之前找到一份有八千块钱工资的、在广州的前端工作。那个时候真的很单纯，啥也不懂，为什么在广州也不过是当时打算和父母住在一起，也可以省下一笔开支。至于为什么要写博客，那时在网上看的视频教学，老师说平时要多写博客，等到面试的时候如果遇到不会的问题，就可以和面试官说「这个点我暂时想不起一些具体细节，但是在我的博客里面我有详细的归纳，您可以看看」这样。

我当时就想，要是我把所有的知识点都写在博客里面，那样一到面试的时候我不是什么难题也不怕了吗？当然，事实证明，并不能一遇到问题就和面试官说要看博客，一来这样感觉确实不太好，二来面试官可能并没有打开你的博客的兴趣，只会觉得你很水。虽然有所怀疑，但是我也决定要好好写博客，得益于在校期间的丰富时间，我原本期望是要在找工作之前写够一百篇之数，这样在面试的时候也可以让别人看到我的一些积累。结果到现在（2019-8-3 15:27:27）入职了都还没有实现这个目标，又要做自己的小项目，又有惰性的原因在里面，最终到找工作的时候只有 80 篇。

我不知道在我求职的过程中，这 80 篇博客有没有起到关键的作用，但是博客确实给我的前端学习添加了许多仪式感，每当看完一个比较重要的知识点，就可以把它写进博客里面，查阅起来也方便。而且博客里的一些话，在面试的时候正好用来回答面试官的问题，倒也没有什么关系。因为这层关系，我也不会再入职之后就卸磨杀驴，完全对博客弃之不顾了（其实也断更了好几个月😂）。现在我打算以后有空的时候还是要把我的博客继续更新一下，比不上以前那样写一篇用两三个小时，一个月写十几篇，我还是希望自己有一周一篇的输出，就这样吧。

这里的博客原来是使用 hexo 「强力驱动」的，后面 hexo 驱动不了了，终于只有放弃。但是博客还得接着开，当时想着之前做个一个 ui 组件库的文档，用的 VuePress 好像还不错，就试着用用看吧。这几天把博客渐渐地从 hexo 搬移到现在这里来，感觉靠谱多了。后面又发现了 VuePress-theme-reco 这个主题，非常好用，完全符合我极简的目的，在这里对这个主题的作者表示衷心的感谢。

最后简单地做一个自我介绍。我是这个博客的作者广宇，南昌大学计算机系本科毕业生，现在上海从事 Web 前端研发工程师的工作。现在自己要完善的地方还有太多太多。我的个人邮箱：1574989439@qq.com，欢迎你的指教。
